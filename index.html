C USTOMGRID Basado en flexbox

CustomGrid fue creada para agilizar la creacion de grillas. Este esta basada en un lenguaje semantico, que la hacen facil de interpretar. Cuenta con mixings para poder crear grillas de manera personalizada y que estas se adapten a tu diseño.

Algunas de las caracteristicas mas importantes son:

  • Soporte flexbox
  • Customizacion de grillas
  • Responsive
  • Desarrollo en sass
  • Implementacion
  • Mixings

Objeto de configuración

CGRID tiene un objeto de configuración definido, desde donde podras customizar.. desde el numero de grillas, el nombre de las clases, el ancho del layout, el espacio entre filas y los breakpoints. 

  
  $settings: (
    global: (
      unit: 'px'
      layout: 1000
    )
    row: (
      class: 'cgrid-row'
    )
    column: (
      total: 12
      class: 'col'
      prefix: 'col-'
    )
    justify_content: (
      class: 'cgrid-j-c'
    )
    align_content: (
      class: 'cgrid-a-c'
    )
    align_items: (
      class: 'cgrid-a-i'
    )
    align_self: (
      class: 'cgrid-a-s'
    )
    direction: (
      class: 'cgrid-d'
    )
    wrap: (
      class: 'cgrid-w'
    )
    spacing: (
      width: 10
    )
    viewports: (
      mobil: 426
      tablet: 800
      desktop: 1200
      wide: 1200
    )
  )

Breakpoints

La definicion de los breakpoints estan basados en 4 medidas.

  Mobile Tablet Desktop Desktop Large
Medidas ≤ 425px ≤ 799px ≤ 1199px ≥ 1200px
Clases + Sufijo [nombre-clase]-xs [nombre-clase]-sm [nombre-clase]-md [nombre-clase]-lg

Mixings

Los mixings nos ayudaran a definir grillas personalizadas. Estos son perfectos para los diseños personalizados. Con los mixings podremos definir atributos en elementos especificos del DOM.

Container

Define ancho del contenedor

  
  @include container($width: $global-layout)
Arguments Value Description
$width input Define el ancho del contenedor

Make row

Agrega atributos

  
  @include make-row($fluid: false, $flex-direction: row, $flex-wrap:wrap)
Arguments Value Description
$fluid false | true Agrega o quita margen negativo
$flex-direction row | row-reverse | column | column-reverse Define tipo de direccion de eje principal
Body
Aside

Make Col

Define ancho de columna por defecto

  
  @include make-col($flex-grow: 1, $fluid: false)
Arguments Value Description
$flex-grow 0, 1.. column-total Número que indica el crecimiento del ítem respecto al resto.
$fluid false | true Agrega o quita padding
Body
Aside

Make Custom Col

Define ancho de columna


@include make-custom-col($percent, $viewport: false, $strict: false)
Arguments Value Description
$percent 0, 1.. column-total Define el porcentaje de la columna
$viewport false | wide | desktop | tablet | mobile Define donde se va aplicar el porcentaje, por defecto es false y se aplica sin viewport.
Body
Aside

Make Custom Offset

Define ancho de offset

  
  @include make-custom-offset($percent, $viewport: false, $strict: false)
Arguments Value Description
$percent 0, 1.. column-total Define el porcentaje del offset
$viewport false | wide | desktop | tablet | mobile Define donde se va aplicar el porcentaje, por defecto es false y se aplica sin viewport.
Body
Aside
